## Getting Started

1. Choose a template from our collection
2. Clone the repository
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```
4. Set up your environment variables
5. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```
